### 📖 项目背景

**Hermes Agent** 是由美国顶尖 AI 研究团队 **Nous Research** 开发的开源 AI 智能体框架。本项目是官方仓库 <NousResearch/hermes-agent> 的社区衍生版本。

我们完整保留了 Hermes 的核心能力（自我进化技能系统、分层持久记忆等）。推出此便携版的初衷，是希望为国内 Windows 用户提供一个更为轻量、低门槛的本地化运行方案。

---

### 🎯 针对 Windows 环境的优化

在本地部署官方原版时，国内用户可能会遇到一些环境适配问题。本便携版针对以下场景进行了封装：

- **本地化运行环境**：官方初始化需拉取海外依赖，国内网络环境下容易超时。本便携版已将核心依赖预置打包，无需特殊网络环境即可直接启动。
- **脱离系统盘限制**：官方默认会将部分运行文件写入 C 盘。本便携版的所有运行文件与用户数据均完全隔离在程序目录内，支持放置在 D 盘、移动硬盘或 U 盘中，不占用系统盘空间。
- **内置完整运行环境**：官方部署需要手动配置 Python、Node.js 等前置依赖。本便携版内置了隔离的虚拟环境，无需在宿主机上安装任何前置包，对初次接触此类工具的用户较为友好。

### ✨ 核心特性

#### 完整继承官方能力
- **自我进化技能**：Agent 可从复杂任务中自动提炼可复用的 Skill。
- **五层分层记忆**：跨会话保存用户偏好与历史，支持全文搜索。
- **多模型支持**：兼容 OpenRouter、DeepSeek、Kimi、OpenAI 等主流 API。

#### 便携版专属
- **U 盘即插即用**：无需管理员权限，不写注册表，不污染系统环境变量。
- **完美兼容中文路径**：针对 Windows 环境优化，解决反斜杠、空格及中文路径导致的报错。
- **数据完全隔离**：所有配置、记忆、技能均保存在 `hermes/` 目录下，复制文件夹即可一键备份。

### 2. 项目介绍文案（适用于发布说明、博客或社区推广）

**Hermes Agent 便携版：一个面向 Windows 用户的本地化运行方案**

**Hermes Agent** 是由美国 AI 研究团队 **Nous Research** 开源的 AI 智能体框架。它拥有自我进化技能和分层记忆系统，是一个非常有潜力的开源项目。

不过，在本地部署官方原版时，国内 Windows 用户往往会遇到一些环境适配上的小麻烦：比如初始化时需要拉取海外源，在网络不佳时容易超时；或者默认情况下部分文件会被写入 C 盘，对系统盘空间不太友好；此外，手动配置 Python、Node.js 等前置依赖，对初次接触这类工具的用户来说也需要花费不少时间。

为了解决这些本地化痛点，我们基于官方代码封装了这个 **Hermes Agent 便携版**。

我们将 Hermes 运行所需的所有组件打包进了一个免安装的绿色包中。它不需要特殊网络环境，完全不占用 C 盘空间，也无需在电脑上安装任何前置依赖。如果你对 Hermes 感兴趣，只需将它解压到 U 盘或本地硬盘，插入任意一台 Windows 10/11 电脑，双击 `start.bat` 即可直接体验。

退出程序后，宿主机不会留下任何痕迹，你的所有记忆和配置都保存在 U 盘内。

如果你刚好需要一个方便随身携带的 AI 助手，或者想在不污染本机环境的前提下体验 Hermes，欢迎随时下载。如果在使用过程中有任何建议，也欢迎在 GitHub 提交 Issue
